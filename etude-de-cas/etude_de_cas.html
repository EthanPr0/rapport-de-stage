Etude de cas


Présentation du système

Afin de pouvoir visualiser le rapport de stage sur internet j'ai réalisé ce site web.

Celui-ci a été développé en Html, Css et Js qui sont des langages web, Html signifie Hypertext Markup Language, Css signifie Cascading Style Sheets et Js signifie Javascript.
Le code html affiche les écritures, les boutons, etc.. le Css affiche le style de la page (couleurs, taille, police, etc..) et le Javascript sert à faire certaines interactions tels que les boutons qui font défiler les pages.
Le site est hébergé sur github car c’est un hébergeur de code gratuit et très utilisé par les développeurs.

Fonctionnement du système

Un site internet est un ensemble de fichiers qui, pour un site internet porte l’extension .html. Ces fichiers vont être stockés sur ce qu’on appelle un serveur d’application lorsque celui-ci sera mis en ligne.



En stockant le site sur ces serveurs, il devient accessible à tous via un navigateur web. C’est d’ailleurs à ces navigateurs, avant les utilisateurs, que ces fichiers s’adressent puisqu’ils sont construits de manière à être lisible par les navigateurs en question via des langages bien spécifiques : Html, Css et Javascript.
Chaque page du site internet défile sur la page index.html lorsque l’on clique sur les boutons correspondants.

Quelques explications du code :

.btn-hover : reprend la class indiqué dans le code html,
.btn-hover:hover : définie l'apparence du bouton au moment où le curseur le survole,
.btn-hover:focus : cible le bouton lorsqu’il est cliqué,
.btn-hover.color : reprend les 2 class utilisés dans le code html,
Width : définie la taille du bouton,
Color : définie sa couleur,
Cursor : indique comment le curseur de la souris sera affiché,
Margin : définie la marge,
Height : définie la hauteur,
Text-align : indique où le texte dois s’aligner,
Border : définie la bordure,
Background-size : définie la taille du fond du bouton,
Border-radius : indique la taille de la rondeur de la bordure,
-o-transition, -webkit-transition, transition : définie l’animation,
Background-position : indique la position du fond animé,
Outline : définié le contour,
Background-image : définie le fond,
Box-shadow : définie l’ombre du bouton.

Eléments du système

Le site est constitué de 10 pages, la page d'accueil qui doit toujours se nommer index.html ainsi que les autres pages qui constituent le rapport. Leurs styles sont déjà fait directement dans les fichiers bootstrap mais certaines pages ont quand même un fichier css à coté car certaines class bootstrap ne fonctionnaient pas bien donc j'ai recodé certaines propriétés. J'ai utiliser les class bootstrap afin qu'elles correspondent au css du fichier.

Installation, mise en service

Problématique: Faire une site web pour le rapport de stage.

Réalisation du site web en Html, Css et Js :

Matériels nécessaires :

  • - Visual studio code,

  • - Les fichiers bootstrap,

  • - Un navigateur,

  • - Un hébergeur.


Procédure :

J'ai dû coder en html, css et js afin de faire un site web regroupant toutes les étapes du rapport de stage.

J'ai commencé par faire l'index.html qui est donc la page d'accueil, pour avoir le corps de la page qui permet
d'afficher le contenu des autres pages sur une seule page afin de gagner du temps et que l'utilisateur n'est pas
besoin de charger chaque pages à chaque fois qu'il change de page.
Toutes les pages sont chargées sur la page principale.

Puis ensuite j'ai codé les autres page une par une. Les pages sont reliées à la pages de style bootstrap grâce à une balise link.



Et pour l'hébergement du site j'ai utilisé github.
